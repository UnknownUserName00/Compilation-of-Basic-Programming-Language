Note:
   For the universal compatibility of your entities that you will use to program for any internet browser, I prefer to use only the entity format and not in a numerical value in decimal format, hexadecimal format, etc. Because some internet browsers tend to use the coded programs that you use to program for your browser and did not use your program to show the right symbol of your program. Example, I will use the Copyright symbol. The code for this in an entity format is &copy; that must show ©. The code in decimal format of it is &#169; that must show ©. If your internet browser shows the right output of your code on it, good for you but some internet browsers are not and tend to use the output of &#169;. So, it is better to use an entity format than on other format(s) either in a decimal or in a hexadecimal format.
