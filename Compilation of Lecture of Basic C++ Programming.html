Definition of Basic Syntax Preparation of a Blank C++ File
C++ Code(s) Description
#include ⇒ Tells the preprocessor to include the iostream standard file. This specific file (iostream) includes the declarations of the basic standard input-output library in C++.
<iostream> ⇒ unknown
using namespace std; ⇒ All the elements of the standard C++ library are declared within what is called a namespace, the namespace with the name std. So in order to access its functionality we declare with this expression that we will be using these entities. This line is very frequent in C++ programs that use the standard library, and in fact it will be included in most of the source codes.
int main () { } ⇒ This line corresponds to the beginning of the definition of the main function. The main function is the point by where all C++ programs start their execution, independently of its location within the source code. It does not matter whether there are other functions with other names defined before or after it - the instructions contained within this function's definition will always be the first ones to be executed in any C++ program. For that same reason, it is essential that all C++ programs have a main function.
return 0; ⇒ The program will return to 0 after the program execution.

Some Statements that Control the Flow of the Program
C++ Code(s) Name Syntax Description
if () { } else if () { } else { } if-else Statement if (name1 ) { // Do Your Codes Here } else if (name1 ) { // Do Your Codes Here } else { // Do Your Codes Here } or if ((name1 ) (name3 )) { // Do Your Codes Here } else if ((name1 ) (name3 )) { // Do Your Codes Here } else { // Do Your Codes Here } ⇒ Used to test your code(s) to run the program under if-else Statement. If the arguments under if Statement is true, the code(s) under that statement will run the program. If it is false, the else Statement will run the program. If an if Statement is declared right beside an else Statement, it will test first all the code(s) outside an else Statement before running that statement together with an if Statement.
while () { } while Loop while (name1 ) { // Do Your Codes Here } or while ((name1 ) (name3 )) { // Do Your Codes Here } ⇒ Used to loop a statement simultaneously if the argument in while () is true. If the argument from that is false, the loop will be terminated.

for () { } for Loop for (name1 = ; name1 ; ) { // Do Your Codes Here } ⇒ Used to loop a statement simultaneously if the argument is false and adds the increment or decrement variable to the initialize variable to make the arguments true and make the loop terminated or use only the increment or decrement variable to the initialize variable to loop the statement continuously until the agreement becomes true.
switch () { case " ": break; case " ": break; default: break; } or switch () { case ' ': break; case ' ': break; default: break; } or switch () { case (): break; case (): break; default: break; } switch Statement switch (name) { case "characters or letters1": // Do Your Codes Here break; case "characters or letters2": // Do Your Codes Here break; default: // Do Your Codes Here break; } or switch (name) { case 'character or letter1': // Do Your Codes Here break; case 'character or letter2': // Do Your Codes Here break; default: // Do Your Codes Here break; } or switch (name) { case (mathematical operation1): // Do Your Codes Here break; case (mathematical operation2): // Do Your Codes Here break; default: // Do Your Codes Here break; } ⇒ Used to get the name of the code declared by the programmer and get it by switch Statement to use and run the inside features of that statement. The switch Statement is just like an if-else Statement. If the first case condition is true, it will run its content code(s) or program. If the first case condition is false, the second case condition will be use to run a program. If it is false again, it will find the case condition to make a statement true to run a program. If all case condition is false, it will fall to default condition. The break Statement is required in each case condition except default condition. If the case condition does not meet the requirement, all case condition not having break Statement will make as one case condition until it reaches a condition that uses a break Statement. If not, the program will flow continously without having that statement until it reaches a default condition.

For the symbols of ' ' and " ":
  • ' ' means one character or letter can be declared in a case condition that is based in a char data type.

  • " " means two or more characters or words can be declared in a case condition that is based in a string declaration.
