 ⇒ Used to get the name of the code declared by the programmer and get it by switch Statement to use and run the inside features of that statement. The switch Statement is just like an if-else Statement. If the first case condition is true, it will run its content code(s) or program. If the first case condition is false, the second case condition will be use to run a program. If it is false again, it will find the case condition to make a statement true to run a program. If all case condition is false, it will fall to default condition. The break Statement is required in each case condition except default condition. If the case condition does not meet the requirement, all case condition not having break Statement will make as one case condition until it reaches a condition that uses a break Statement. If not, the program will flow continously without having that statement until it reaches a default condition.

For the symbols of ' ' and " ":
  • ' ' means one character or letter can be declared in a case condition that is based in a char data type.

  • " " means two or more characters or words can be declared in a case condition that is based in a String declaration.
